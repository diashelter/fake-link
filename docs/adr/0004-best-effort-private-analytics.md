# Analytics privados e best effort

Analytics serão privados, minimizados e processados sem bloquear um redirect válido; falhas podem causar perda de métricas, nunca indisponibilidade deliberada do link. Estimated Unique Clicks usarão identificação pseudônima diária sem persistir IP ou user-agent brutos, aceitando imprecisão para reduzir rastreamento e exposição de dados pessoais.
